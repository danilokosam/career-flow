# Career-flow

Career-flow is a web app to help users track and manage their job application process in one place.

The project is currently in progress. At this stage, the main focus is building a clean dashboard experience with core job-tracking features, including search, filtering, pagination, and basic analytics views.

## Current Scope

- Job application management (create, edit, list, and review entries)
- Search and status-based filtering
- Dashboard views with stats and charts
- Authentication flow and guest/test access support

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- TanStack Query
- Prisma

## Run Locally

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
